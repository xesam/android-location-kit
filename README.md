# 地图定位装饰

## 槽点

### 共同槽点

都没有超时设置，只能干等定位回调，或者手动中断定位。

### 百度

方法命名特别恶心，随意列举几个例子：

    BDLocation#getTime()

返回的居然是字符串，不知道程序员怎么想的，既不符合约定，也非常不便于后续处理！


    BDLocation#getRadius()

获取定位精度，这个名字取得也是没谁了。类似的命名真是太多了。

单次定位与连续定位的纠缠不清

如果 App 同时存在单次定位与连续定位的需求，那么在连续定位开启的期间，如果发起单次定位，是不是会也触发连续定位的回调，即使这个时候根本没有到连续定位的触发时间点。
